# [<-](../README.md) Teriyaki

## Incredients
| Incredients |              |
| ----------  | ------------ |
| 15g         | Corn starch  |
| 100ml       | Water        |
| 100ml       | Soy sauce    |
| 100ml       | Sake         |
| 50g         | Brown sugar  |
| 20g         | Sesame oil   |
| 1-2 cloves  | Garlic       |


## Method

### 1. Step
Grate the garlic very finely.

### 2. Step
Mix everything well. Heat while stirring.
