# [<-](../README.md) Cheese Cake Souffle

## Incredients
| Incredients |              |
| ----------  | ------------ |
| 150ml       | Cream Cheese |
| 3           | Eggs         |
| 50g         | Milk         |
| 65g         | Erythritol   |
| 10g         | Corn starch  |
| 25g         | Flour        |
| 2g          | Salt         |
| 8g          | Lemon juice  |

### Notes
- 140 -> 40min
- Turn off and wait for 50 maybe 40 degree cooldown. 
- Overbeat egg whites. 
- Salt and lemon is critical.
